Cuestionario Sobre el contenido de la Unidad I.

11. Cuál es la función básica de un Sistema Operativo

Actúa como un intermediario entre el hardware y el usuario, gestionando los recursos del sistema y permitiendo la ejecución de aplicaciones.

12. Mencione tres Sistema Operativo más utilizados

Los tres sistemas operativos más utilizados para computadoras son Microsoft Windows, Mac OS y Linux:

  • Microsoft Windows: Es uno de los sistemas operativos más populares y extendidos, especialmente para PC. Windows 10 es la versión más preferida.
  • Mac OS: Es el sistema operativo de los ordenadores Apple.
  • Linux: Está disponible en varias versiones con un funcionamiento similar y también es para PC.

13. Menciona algunas tareas que realizan los Sistemas Operativos

  • Administrar recursos de hardware: Como el procesador, la memoria, el disco duro, la red, el teclado, el mouse, la impresora y la placa de red.
  • Gestionar la memoria: Para que cada programa pueda tener una entrada de memoria independiente, incluso cuando varios programas se estén ejecutando.
  • Ejecutar aplicaciones: Designando los recursos necesarios para cada aplicación.
  • Garantizar la seguridad: Impidiendo el acceso a ciertos archivos o programas, y aplicando restricciones de acceso para que los programas y aplicaciones
    solo puedan acceder a las partes de la memoria que necesiten.
  • Detectar errores: Para mantener la operatividad y controlar dispositivos, y evitar interrupciones.
  • Administrar usuarios: Permiten que varios usuarios ejecuten simultáneamente sus programas, accediendo a la vez a los recursos de la computadora.

14. Qué es un sistema operativo multitareas

Un sistema operativo multitarea, también conocido como multitasking, es capaz de ejecutar más de un proceso o aplicación al mismo tiempo. Para ello,
divide el tiempo de procesador disponible entre los procesos que lo necesitan, asignando un segmento de tiempo a cada subproceso que se está ejecutando.

15. Qué es un sistema operativo multiusuario

Un sistema operativo multiusuario es un programa o sistema operativo que permite que múltiples usuarios puedan recibir servicio y procesamiento al mismo tiempo.
Este tipo de sistema operativo se utiliza a menudo en empresas, escuelas y otras organizaciones que necesitan proporcionar acceso al mismo ordenador o programa
de software a varias personas a la vez.

16. Cómo funciona un sistema operativo

El SO actúa como intermediario entre el usuario y el hardware, ofreciendo un entorno necesario para que el usuario pueda ejecutar sus programas.

17. Cómo se utiliza un sistema operativo en un entorno gráfico

Un sistema operativo en un entorno gráfico utiliza una interfaz gráfica de usuario (GUI) para permitir la interacción visual entre el usuario y el dispositivo electrónico.
La GUI es una pieza fundamental en un entorno gráfico y es una evolución de las interfaces de línea de comandos que se usaban en los primeros sistemas operativos.

18. Cuándo inicia el funcionamiento del Sistema Operativo

El sistema operativo se inicia cuando se enciende un ordenador y se presiona el botón de encendido. Durante el proceso de arranque, el sistema operativo carga y ejecuta el
sistema operativo, prueba el hardware y configura los dispositivos.

19. Mencione las dos interfaces que forman parte del Sistema Operativo

Las dos interfaces que forman parte del sistema operativo son la interfaz de línea de comandos (CLI) y la interfaz gráfica de usuario (GUI):

  • Interfaz de línea de comandos: Es alfanumérica, es decir, solo presenta texto.
  • Interfaz gráfica de usuario: Representa visualmente los elementos de control y medida para optimizar la comunicación con el usuario y, en general, la experiencia de este.
    Es usada en la mayoría de sitios web, programas para computadora, aplicaciones móviles, sistemas operativos, entre otros tipos de software existentes.

20. Qué es un proceso en entorno de los sistemas operativos

En un entorno de sistemas operativos, un proceso es un programa en ejecución que incluye el valor del contador de programa, los registros y las variables.
Los sistemas operativos utilizan los procesos para separar las aplicaciones que se están ejecutando.
