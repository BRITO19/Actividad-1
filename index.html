Cuestionario Sobre el contenido de la Unidad I.

1. Qué es un Sistema Operativo

Un sistema operativo (SO) es un conjunto de programas que permiten manejar los recursos de una computadora, como la memoria, el disco, los dispositivos
de entrada/salida y los periféricos, como el teclado, el mouse y la impresora.

El SO también permite la comunicación entre el usuario y la computadora, ya que es la interfaz que conecta los dispositivos periféricos con la computadora.

2. Componentes de un Sistema Operativo

Los componentes principales de un sistema operativo son:

  • Kernel o núcleo: Gestiona los recursos del hardware y proporciona servicios a los programas.
  • Interfaz de usuario: Permite la interacción entre el usuario y el sistema operativo, como interfaces gráficas o de línea de comandos.

3. Tareas del Sistema Operativo

  • Implementar diferentes entornos para diferentes usos (interfaz gráfica, Shell, tipo web, etc.)
  • Proveer una o más interfaces con el usuario.
  • Proveer a las aplicaciones un conjunto de servicios (a través de los “system services”)
  • eficiencia y equidad en la administración de recursos.

4. Funciones de un Sistema Operativo

  • Asigna y administra la memoria RAM entre los diferentes programas.
  • Coordina y prioriza las tareas de la CPU.
  • Organiza y controla el acceso a los datos almacenados.
  • Gestiona la comunicación con hardware como impresoras, cámaras o discos duros.
  • Proporciona una interfaz gráfica o de línea de comandos para que el usuario pueda interactuar con él y desarrollar las actividades que necesita en cada momento.

5. Características de un Sistema Operativo

  • Es el intermediario entre el usuario y el hardware.
  • Es necesario para el funcionamiento de todos los computadores, tabletas y teléfonos móviles.
  • Otorga seguridad y protege a los programas y archivos del ordenador.
  • Está diseñado para ser amigable con el usuario y fácil de usar.
  • Permite administrar de manera eficiente los recursos del ordenador.
  • La mayoría requiere del pago de una licencia para su uso.
  • Permite interactuar con varios dispositivos.
  • Es progresivo, ya que existen constantemente nuevas versiones que se actualizan y adaptan a las necesidades del usuario.

6. Tipos de Sistema Operativo

Los sistemas operativos se pueden clasificar en diferentes tipos según su uso, como, por ejemplo:

  • Sistemas operativos de escritorio
  • Diseñados para ordenadores personales y profesionales, como Windows, macOS y Linux.
  • Sistemas operativos móviles
  • Diseñados para dispositivos móviles, como smartphones, tablets y relojes inteligentes, como Android, iOS, webOS y watchOS.
  • Sistemas operativos de servidor
  • Utilizado en servidores para gestionar redes y recursos compartidos,como Solaris.
  • Sistemas operativos embebidos
  • Presente en dispositivos especializados como routers, smart TVs y sistemas de control industrial.

9. Sistema Operativo según el Ordenador y el Tipo de Aplicaciones

Los sistemas operativos se pueden dividir en cuatro grandes grupos, según el tipo de dispositivo en el que se utilizan:

  • Sistemas operativos de escritorio: Diseñados para ordenadores personales y profesionales, como Windows, macOS y Linux
  • Sistemas operativos móviles: Dominados por Android e iOS para smartphones y tablets.
  • Sistem operativos de servidor: Utilizados en servidores para gestionar redes y recursos compartidos.
  • Sistem operativos embebidos: Presentes en dispositivos especializados como routers, smart TVs y sistemas de control industrial.

10. Compatibilidad de los Sistema Operativo

La compatibilidad del sistema operativo o (o los sistemas operativos compatibles) no está definida por el procesador escalable Intel® Xeon® o el chipset de un servidor, pero
depende de los sistemas operativos compatibles con la placa base o el sistema del servidor. El fabricante de la motherboard o del sistema validará ciertos sistemas operativos
con la motherboard o el sistema y proporcionará controladores para los sistemas operativos compatibles.

La compatibilidad se relaciona con la integración de los sistemas. Para que la información pueda ser compartida y para que los usuarios tengan acceso a sistemas semejantes
con interfaces comunes, los sistemas tienen que ser compatibles. En este contexto la compatibilidad significa compartir equipos, sistemas operativos y programas.
